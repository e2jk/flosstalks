About FLOSS Talks

There are so many interesting projects out there that it can sometimes be difficult to discover new ones. By providing a central place to get audio and video content about FLOSS projects, we hope to contribute to the promotion of these projects.

History of FLOSS Talks

FLOSS Talks was started by Emilien Klein. While preparing a presentation about FreeNAS for our monthly Den Bosch LUG meeting, I searched for a list of podcasts that would present that same project. As I didn't encounter it, I decided to create it ;)

Technical information

The website is created using Django. It is mainly written in Python and uses HTML5 Boilerplate / Initializr / Bootstrap. The icons used on the site are graciously provided by Glyphicons.
